# Model Context Protocol (MCP) Research Project

This repository contains an academic research project on the **Model Context Protocol (MCP)**, a standard for connecting AI assistants to external systems.

## Introduction
The Model Context Protocol (MCP) is described in the official Anthropic documentation: https://docs.anthropic.com/en/docs/model-context-protocol. MCP defines a structured way for AI models to exchange context information with external services, enabling richer interactions and integration capabilities.

The goal of this project is to explore the specifications, implementations, and real‑world use cases of MCP.
